# Garden Advice Program

This project provides gardening advice based on the season and plant type entered by the user.

## Features
- Provides gardening advice based on the selected season
- Provides advice based on the plant type
- Recommends plants that grow in each season
- Uses dictionaries for storing advice
- Uses functions for organisation
- Has comments and docstrings.

## How to Run the Program

1. Open a terminal.
2. Navigate to project folder.
3. Run the program with:

```
python3 garden_advice.py
```

4. Enter the season and plant type when prompted.

Example:

```
Enter a season (summer, winter, spring, autumn): summer
Enter a plant type (flower, vegetable, herb): flower
```

The program then displays: gardening advice and recommended plants.
